# Component, Koa & friends boilerplate

## Installation

### Requires following tools on PATH
* Node >= 0.11.9
* NPM

### Windows
1. Hit setup.cmd (may need to be run as administrator as it contains npm global installs)
1. Hit devel.cmd
1. Success

### unix and derivatives
1. `./setup.sh`
1. `./devel.sh`
1. Success
